Tênis in SP é um projeto de loja virtual fictícia desenvolvido para demonstrar uma plataforma de venda de tênis multimarcas. O site apresenta modelos de algumas das principais marcas do mercado brasileiro, com foco em uma navegação simples, design moderno e experiência intuitiva para o usuário. O projeto foi criado com o objetivo de praticar desenvolvimento web, organização de produtos e construção de interfaces voltadas para o comércio eletrônico.
